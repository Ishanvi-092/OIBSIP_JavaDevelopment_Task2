# Task 2 – Guess The Number Game

## Objective

The objective of this project is to develop an interactive **Guess the Number game using Java** where the system generates a random number and the user attempts to guess it. The program provides feedback to guide the user by indicating whether the guessed number is higher or lower than the generated number.

The project aims to demonstrate basic programming concepts such as **loops, conditional statements, random number generation, and user interaction**.

---

## Tools & Technologies Used

* **Java** – Core programming language used to implement the game logic
* **Java Swing** – Used to display input and message dialog boxes
* **VS Code** – Development environment used for coding and testing

---

## Steps Performed

1. **Generated a Random Number**

   * Used the Java `Random` class to generate a number within a specified range (1–100).

2. **Prompted User Input**

   * Displayed a dialogue box prompting the user to enter their guess.

3. **Implemented Hint Logic**

   * Compared the guessed number with the generated number and displayed messages indicating whether the guess was too high, too low, or correct.

4. **Added Attempt Limitation**

   * Restricted the number of guesses per round to increase the difficulty of the game.

5. **Implemented Multiple Rounds**

   * Allowed the user to play multiple rounds of the game.

6. **Added a Scoring System**

   * Assigned points based on the number of attempts taken to guess the number.

7. **Tested the Game**

   * Verified that hints, attempts, and score calculations worked correctly.

---

## Outcome

The project successfully demonstrates a simple **number guessing game** where users interact with the system to identify a randomly generated number. The game provides hints, tracks attempts, and calculates scores across multiple rounds.

Through this project, practical experience was gained in **random number generation, control structures, loops, and user interaction in Java**.
